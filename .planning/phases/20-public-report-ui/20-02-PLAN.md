      - Show LCP, INP, CLS as three prominent metric cards

      - INP: "Interaction to Next Paint" — value in ms ("200ms")
      - CLS: "Cumulative Layout Shift" — unitless (show 2 decimals: "0.10")
    - **Mobile/Desktop scores:**
      - Show as circular score indicators or simple numeric displays (0-100)
      - Color code: 90-100 green, 50-89 amber, 0-49 red
    - **Optional image audit:** If imageAudit present, show issue count with severity
    - Use semantic HTML: `<section>`, `<h2>`, `<dl>` for metric pairs

      - OG Tags: show count of tags found, or list key ones (og:title, og:image)
      - Canonical URL: show URL or "Not set"

      - Robots Meta: show value or "Not set"

      - Sitemap: checkmark if found, X if not
      - Structured Data: count of JSON-LD schemas found

      - CDN: badge if detected
      - Payment Providers: badges
      - Chat Tools: badges
      - Review Widgets: badges
    - Use a compact badge-based layout — each detected tool is a pill/badge
    - Group into rows: Platform (large), then rows for each category

    - **Loading state:** Show a skeleton/loading indicator while data loads
    - **Error state:** If session not found (404), show "Report not found" message
