## MCP é o padrão para acesso a ferramentas de IA. Em escala, você precisa de mais do que conexões — precisa de trilhas de auditoria, controles de acesso e logging unificado.

**deco CMS é a plataforma de gerenciamento de contexto que faz o MCP funcionar para equipes.**

É o control plane para tráfego MCP — registre cada servidor MCP uma vez com credenciais centralizadas, e ele se torna disponível (com controles de acesso e logging de auditoria adequados) em todos os seus projetos, agentes e fluxos de trabalho.

### O Problema

Conforme agentes de IA se tornam infraestrutura essencial, gerenciar seu acesso cria desafios críticos:

- **Proliferação de conexões**: Cada funcionário configura manualmente conexões MCP em seu laptop
- **Sem trilhas de auditoria**: Ações de agentes invisíveis para sistemas de logging e conformidade
- **Segurança fragmentada**: Sem controle centralizado sobre quem acessa o quê

Em escala empresarial, equipes ou abandonam a integração MCP interna, operam com permissões perigosamente amplas ou gastam meses construindo infraestrutura personalizada.

### Como o deco CMS Resolve

**Conecte**: Registre cada servidor MCP uma vez. Credenciais armazenadas com segurança no vault, disponíveis em toda a organização. Atualize API keys uma vez, não em cada máquina.

**Orquestre**: Controle de acesso granular com RBAC. Controle quem usa quais ferramentas, quando e como. Trilha de auditoria completa mostrando identidade do chamador, nome da ferramenta, entrada/saída e timing.

**Remixe**: Combine e filtre MCPs em MCPs Virtuais em tempo de execução — sem deploy de código. Crie versões somente leitura, agregue ferramentas de múltiplas fontes ou construa coleções de ferramentas específicas por papel.

### Principais Funcionalidades

- **Autenticação centralizada**: SSO/OAuth 2.1, API keys, vault de credenciais criptografado
- **Logging de auditoria unificado**: Toda requisição MCP registrada com contexto completo
- **MCPs Virtuais**: Superfícies de ferramentas editáveis em tempo de execução sem deploy de código
- **decopilot**: Orquestração integrada para fluxos de trabalho agênticos multi-etapas
- **Agnóstico de modelo**: Funciona com Claude, OpenAI ou qualquer provedor de IA

### Para Quem É

- **Empresas** gerenciando acesso de agentes de IA a sistemas internos (GitHub, Slack, bancos de dados, APIs)
- **Equipes** executando fluxos de trabalho agênticos que precisam de orquestração, logging e controle de acesso
- **Equipes de segurança** que precisam de trilhas de auditoria e relatórios de conformidade para sistemas de IA
